1. border: 简写属性, 可同时定义border-width, border-style, border-color
  2. border-width: 边框宽度, 包含border-top-width, border-right-width, border-bottom-width, border-left-width, 可取以下四种值
    1. length: 用长度来定义边框的厚度, 不允许为负数
    2. medium: 定义默认厚度的边框, 计算值为3px
    3. thin: 定义默认厚度细的边框, 计算值为1px
    4. thick: 定义默认厚度粗的边框, 计算值为3px
  3. border-style: 边框样式, 包含上 右 下 左四个子属性, 其值可取下列属性值:
    1. none: 无轮廓, 当定义了该值时, border-color将被忽略, border-width计算值为0, 除非边框轮廓应用了border-image
    2. hidden: 隐藏边框
    3. dotted: 点状轮廓
    4. dashed: 虚线轮廓
    5. solid: 实线轮廓
    6. double: 双线轮廓, 两条单线与其间隔的和等于指定的border-width值
    7. groove: 3d凹槽轮廓
    8. ridge: 3d凸槽轮廓
    9. inset: 3d凹边轮廓
    10. outset: 3d凸边轮廓
  4. box-shadow: 定义元素的阴影: inset? && <length>{2,4} && <color>?, 值的定义如下
    1. none: 无阴影
    2. length-1: 第1个长度值定义元素的阴影水平偏移量, 正值, 阴影出现在元素右侧; 负值, 则阴影出现在元素左侧
    3. length-2: 第2个长度值定义元素的阴影垂直偏移量, 正值, 阴影出现在元素底部, 负值, 则阴影出现在元素顶部
    4. length-3: 第3个长度值定义元素的阴影模糊值半径(如果提供了). 该值越大阴影边缘越模糊, 若该值为0, 阴影边缘不出现模糊. 不允许负值
    5. length-4: 第4个长度值定义元素的阴影外延值(如果提供了). 正值, 阴影将向四面扩展; 负值, 则阴影向里收缩
    6. color: 定义元素阴影的颜色, 未定义则阴影颜色将默认取当前最近的文本颜色
    7. inset: 定义元素的阴影类型为内阴影. 该值为空时, 则元素的阴影类型为外阴影.
  5. box-shadow可以定义多组阴影效果. 每组参数值以逗号分隔, 该属性可以对几乎所有元素的生成框产生阴影. 如果元素同时设置了border-radius, 阴影也会有圆角效果
  6. 如果定义了多组阴影, 它们的z-ordering为, 第一个阴影在最上面, 以此类推
