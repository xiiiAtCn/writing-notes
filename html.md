#### link标签的特殊使用
* dns
```html
<link rel="dns-prefetch" href="//g.alicdn.com">
```
* iPhone图片
```html
<link href="//img.alicdn.com/tps/i3/T1OjaVFl4dXXa.JOZB-114-114.png" rel="apple-touch-icon-precomposed">
```
* 网站图标
```html
<link href="/favicon.ico" type="image/x-icon" rel="shortcut icon">
```
