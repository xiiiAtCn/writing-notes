test部分属性
  1. width, min-width, max-width, height, min-height, max-height, 均适用于除非置换内联元素, table-row, table-row-group之外的所有元素
  2. 其中, 六者均可使用<length>或<percentage>, width和height还可以使用auto
  3. auto, 无特定值, 取决于其它属性值
  4. length, 用长度值来定义, 不允许出现负值
  5. percentage, 用百分比来定义宽度, width百分比参照包含块宽度, height百分比需要区别对待
  6. width, min-width与max-width之间的关系
    1. 当min-width属性的值小于width时, min-width属性将会被忽略
    2. 当max-width属性的值小于width时, max-width属性将会被忽略, 同时width会忽略自己的值定义而使用max-width的值作为自己的使用值
    3. 当min-width属性的值大于width时, min-width属性将会被忽略, 同时width会忽略自己的值定义而使用min-width的值作为自己的使用之
    4. 当max-width属性的值大于width时, max-width属性将会被忽略
    5. 当min-width属性的值大于max-width时, max-width属性将会被忽略
    6. 当max-width属性的值小于min-width时, max-width属性将会被忽略
